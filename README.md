## ✨ Features

- 🌍 **Bilingual** — Full English / Arabic (عربي) toggle with instant RTL layout switching
- 📷 **Real Photography** — High-quality luxury home & interior photos
- 🎬 **Scroll Animations** — Elements reveal smoothly as you scroll down the page
- 🖱️ **Custom Cursor** — Gold cursor dot with a lagging ring that reacts to hover
- 📰 **Animated Marquee** — Infinite scrolling services ticker
- 🗂️ **Portfolio Grid** — Asymmetric 12-column layout with hover overlays
- 💬 **Auto-Cycling Testimonials** — Rotating client quotes in both languages
- 📬 **Contact Form** — Fully styled enquiry form with RTL-aware inputs
- ⚡ **Zero Dependencies** — No libraries or frameworks — loads instantly

---

## 🌍 Arabic / RTL Support

Click the **EN / عربي** toggle (bottom-right corner) to switch languages. The entire page updates instantly including:

- All text content and labels
- Form placeholders and select options
- Layout direction (LTR ↔ RTL)
- Navigation order
- Testimonial quotes
- Button arrow direction (→ ↔ ←)

## 🎨 Design Decisions

- **Color palette** — Deep charcoal `#0A0A0A` with warm gold accents `#C8A96E`
- **Typography** — Cormorant Garamond (display) + Montserrat (body) + Noto Naskh Arabic
- **Aesthetic** — Luxury editorial — refined minimalism with intentional use of negative space
- **Animations** — CSS-only where possible; JS only for cursor, testimonials, and language switching
